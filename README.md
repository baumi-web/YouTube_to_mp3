Mit diesem kleinen Programm kann man Youtube-Videos als MP3 speichern. 
Dazu muss man nach Starten des Skripts nur den Link in der Konsole eingeben und nach Aufforderung den gewÃ¼nschten Dateinamen der MP3.
## Requirements
> [!NOTE]
> #### install ffmpeg:
>  https://ffmpeg.org/download.html <br> <br>
>
> #### install  yt-dlp in your envirement
> ```
> pip install yt-dlp
> ```
> or
> ```
> python3 -m pip install yt-dlp
> ```


> [!TIP]
> #### run with:
> ```
> python Link_to_MP3.py
> ```
